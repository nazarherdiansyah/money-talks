Koneksi OneDrive

Langkah 1 dari 4
1

Buat akun GitHub & upload file

Untuk hosting web app secara online — gratis selamanya

Jika sudah punya akun GitHub dan file sudah di-upload, langsung klik Lanjut.
a

Buka github.com → daftar akun gratis → buat repository baru bernama dompet-bersama (pilih Public, centang Add a README file).

b

Rename file HTML yang sudah diunduh menjadi index.html, lalu upload ke repository: klik Add file → Upload files → drag & drop → Commit changes.

c

Aktifkan GitHub Pages: di repository → Settings → Pages → Source → pilih branch main → Save. Tunggu 2 menit, web app online.

2

Daftar aplikasi di Azure

Supaya web app bisa akses file Excel di OneDrive Anda

a

Buka portal.azure.com → login dengan akun Microsoft (Outlook/Hotmail) yang sama dengan OneDrive Anda.

b

Di kolom pencarian atas ketik App registrations → klik → klik tombol + New registration.

c

Isi formulir pendaftaran:

Name: DompetBersama
Supported account types: Personal Microsoft accounts only
Redirect URI: pilih Single-page application (SPA) → isi URL GitHub Pages Anda:
https://USERNAME.github.io/dompet-bersama

Klik Register.

d

Setelah terdaftar, Anda akan melihat halaman seperti ini. Salin Application (client) ID — ini yang akan ditempel di langkah 4:

Azure Portal — App Overview
Display nameDompetBersama
Application (client) ID a1b2c3d4-xxxx-xxxx-xxxx-ef1234567890
Directory (tenant) IDxxxxxxxx-xxxx-...
👆 Klik ikon copy di sebelah kanan ID tersebut, lalu simpan
3

Buat file Excel di OneDrive

File ini jadi database — semua transaksi tersimpan di sini

a

Buka onedrive.live.com → login dengan akun Microsoft yang sama.

b

Klik + New → Excel workbook. Beri nama persis seperti di bawah (huruf besar/kecil harus sama):

DompetBersama.xlsx

Tidak perlu isi apapun di dalam file — web app akan otomatis membuat kolom dan sheet yang dibutuhkan.

4

Tempel Client ID & hubungkan

Langkah terakhir — isi dua kolom di bawah ini

Di sinilah Anda menempel Client ID yang sudah disalin dari Azure (langkah 2d). Copy-paste ke kolom pertama di bawah.

Contoh: a1b2c3d4-e5f6-7890-abcd-ef1234567890

Biarkan seperti ini jika nama file sudah DompetBersama.xlsx
